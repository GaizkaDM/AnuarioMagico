Welcome to the Hogwarts Yearbook. This application allows you to explore, manage, and discover detailed information about characters from the Harry Potter universe. Whether you are a first-year student or a seasoned Auror, this tool is for you.

üîçExploring Characters

The main screen is your window to the magical world. Here you will find:

  • Search: Use the left sidebar to find characters by name or Patronus.
  • Filters: Filter by House (Gryffindor, Slytherin, etc.) or Status (Alive, Deceased).
  • Favorites: Check the "Favorites Only" box to see your preferred characters.
  • Navigation: Use the "Previous" and "Next" buttons or type a page number to navigate quickly.

üìñCharacter Details

Click the "View Details" button on any card to see the character's full profile. In this view you can:

  • View a high-resolution image.
  • Consult biographical data such as wand, patronus, hair color, etc.
  • Export the individual profile to PDF.
  • If you are logged in, you can Edit or Delete the character.

‚òÅÔ∏èSync & Offline Mode

The app is designed to work always, even without internet:

  • Auto Sync: Upon startup, the app attempts to download the latest data from the central database.
  • Offline Mode: If there is no connection, you will work with the local copy of the data.
  • Sync Button: Use this button in the left sidebar to force a manual update of data and images.

üîêUser Management

To add, edit, or delete characters, you need to log in:

  • Go to the menu File > Login.
  • If you don't have an account, you can register (requires a master password provided by the administrator).
  • Once inside, the "Add Character" button will appear in the sidebar.

üìÑReport Generation

Need a printed list?

  • Use the "Generate PDF" button in the sidebar to create a list of the characters you are currently viewing (based on your filters).
  • You can also generate an individual PDF from each character's detail view.
Magical Tip: Enable "Dark Mode" in the View menu for a more comfortable reading experience at night or to feel like you're in the Slytherin dungeons.
